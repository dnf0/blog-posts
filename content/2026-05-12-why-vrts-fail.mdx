# The Problem
Global VRTs degrade at scale due to XML parsing overhead and memory bloat.

# The 6 Alternatives
- Local VRTs
- GeoParquet Direct Reads
- Kerchunk
- Zarr
- Lance Format
- STAC + TiTiler
